# Proyecto Sirenas

Situación de Aprendizaje para Educación Infantil de 5 años sobre literatura asturiana, escritura emergente, juego verbal y trabajo cooperativo.

## Ver la web

La entrada principal del proyecto es:

https://analopezescobar.github.io/mis-cosas-del-cole-/

## Contenido

- Intención educativa y justificación del proyecto.
- Elementos curriculares.
- Metodologías activas.
- Rúbricas e instrumentos de evaluación.
- Actividades complementarias y atribución.

## Estructura

Esta web es una exportación de eXeLearning. Para que funcione correctamente en GitHub Pages deben subirse juntas estas carpetas y archivos:

- `index.html`
- `html/`
- `content/`
- `idevices/`
- `libs/`
- `theme/`
- `content.xml`
